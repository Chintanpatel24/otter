# Contributing to Otter v1.0.0

1. Fork the repository `Chintanpatel/otter`.
2. Create a feature branch from main.
3. Make changes following the clean, minimal design rules (no emojis, matte black dark theme, pure white light theme).
4. Build with `make` and test with `cargo build --release`.
5. Ensure no new decorative elements are added.
6. Test installations using the official one-liner commands:
   - macOS/Linux: `curl -fsSL https://raw.githubusercontent.com/Chintanpatel/otter/main/scripts/install.sh | bash`
   - Windows PowerShell: `iwr -useb https://raw.githubusercontent.com/Chintanpatel/otter/main/scripts/install_windows.ps1 | iex`
7. Submit a pull request using `.github/PULL_REQUEST_TEMPLATE.md`.
